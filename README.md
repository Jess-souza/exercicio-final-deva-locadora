# exercicio-final-deva-locadora
### Projeto final do módulo de Programação Orientada a Objetos I do programa Programa <Dev>ª | Java - da Let's code from Ada :octocat: :heavy_check_mark:

**Situação problema:**
O dono de uma locadora percebeu que alguns filmes do seu estoque não estão mais sendo alugados. Para liberar espaço no estoque e comprar novos CD's e DVD's resolveu doar os que estão parados apenas ocupando espaço. Crie um sistema que faça a doação dessas mídias de forma aleatória.
  
  - Classe pai: midia
  - Classe filha: DVD
  - Classe filha: CD
  - Classe DoacaoMidia
